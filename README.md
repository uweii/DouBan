## 简豆
> 基于豆瓣Api的客户端，包含电影分类，图书分类，电影榜单，收藏功能，缓存功能，简易切换主题功能，每日图片功能。

## 特点
* 更小的APK : APK大小仅为2.72MB
* 贴近Google Material Design 设计规范
